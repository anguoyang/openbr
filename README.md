**www.openbiometrics.org**

Identify the latest [tagged release](https://github.com/biometrics/openbr/releases).

    $ git clone https://github.com/biometrics/openbr.git
    $ cd openbr
    $ git checkout <tag>
    $ git submodule init
    $ git submodule update
